通信系统一般模型

A.系统模型(以电通信为例子)
->发信者->(模拟/数字)->发送器->(模拟)->信道->(模拟)->接收器->(模拟/数字)->收信者->(恢复的消息)

(其中:()噪声源)->信道)
发信者 : 信息源(简称 信源),各种消息转换成原始电信号(电通信)
收信者 : 信宿,原始电信号还原成相应的消息(电通信)
发送器 : 基本功能是jiangxinyuanhexindao

B.关于信源

  • 信源,消息,信号,信息
    • 信源 : 消息的发源地,
    • 消息 : 由信源产生,通信系统要传送的具体内容
    • 信号 : 是消息的载体
      -
  • 数字信源和模拟信源
    -
    • 模拟信源:信息是由无限多种取值的连续量
  • 常见的消息类型
  • 文字消息
    -
    • 语音
      • 300~3400Hz
    • 音乐
      • 20Hz~15(20)Hz
    • 图像
      -
    • 数据
      C.典型通信系统(广播电台)
  • 发射端
  • 接收端

模拟通信系统模型和数字通信系统模型

A 模拟通信系统

传输模拟消息的通信系统
分为 : 模拟基带通信系统和模拟带通通信系统

模拟带通
消息信号->调制->发射机->传输媒质->接收机->解调->收到的消息信号
调制 : 借助于正弦载波形形成频带信号
S(t) = R(t)cos[2Πf_ct+θ(t)]
解调 : 逆向

B 数字信号系统

传输数字消息的通信系统
语音->格式化->信源编码(压缩)->加密->信道编码->复用(多用户)->脉冲基带调制->频带调制频道扩展->多址接入->发射机->信道->(逆向)

通信系统主要性能指标

通信系统的基本性能指标

  • 有效性
    • 指在给定信道内能传输的信息的多少
    • 模拟通信系统:用信号的有效传输频带来度量
    • 数字通信系统:用传输速率和频带利用率来度量
    • 频带 : 带宽
  • 可靠性
    • 指接受信息的准确程度
    • 模拟通信系统:用接收端解调输出信噪比来度量
    • 数字通信系统:用误比特率或误符号率来度量

      模拟通信系统的基本指标

  • 有效性: 频带的宽度模拟小孩
  • 可靠性:

    数字通信系统的基本指标

  • 有效性:传输速率,用比特率R_b或波特率R_B表示
    • 比特率: bit/s(bps)
    • 波特率:每秒内系统传送的符号数(码元数),单位为波特(baud)
    • 关系 R_b = R_Blog_2M,一个码元用M个比特表示,
    • 利用率: 单位带宽的传输速率,即μ=R_B/B(B/Hz)或μ_b=R_b/B(bps/Hz)
  • 可靠性:误比特率P_b或误符号率P_e
    • 二进制传输中 P_b = P_e
