第四章 信道

发表于 2018-09-06

信道

信道 : 介于发信者和收信者之间,供消息经过的通道,
频带 :
信道频带:信道所能传送的信号频率范围
信号频带:信号所含的频率与所占的带宽
确定信号:有频谱分析得到;随机信号:由功率谱分析得到

无线信道

在无线信道中信号的传输是利用电磁波在空间的传播
原则上,任何频率的电磁波皆可产生,但为了有效发射或接收电磁波,要求:天线的尺寸不小于电磁波波长的1/10
因此,频率过度,波长过长,则天线过长,难于实现

电磁波传播的主要模式

  • 地波模式
  • 天波模式
    • 电磁波经电离层反射而折回地面的传播方式,一次反射可传约4000km
  • 视线模式
    • 直线传播,有最大视距

有线信道

  • 明线
  • 对称电缆(双绞线) 损耗比
  • 同轴电缆
  • 光纤

信道的数学模型

  • 狭义信道 (有线信道,无线信道)
  • 广义信道
    • 调制信道
    • 编码信道

调制信道模型

调制信道有一对输入输出端,关系表示为:
e_0(t) = k(t)e_i(t)+n(t)
一般为时变模型,n(t)为加性噪声(或干扰)

信道特性对信号传输的影响

  • 相位~频率特性:要求其为通过原点的直线即群时延为常数时无失真

恒参信道特性分析

  • 系统的传输函数h(t)=klandau(t-tao)的FT:
  • 时域分析:信道输出是输入的时延和缩放
  • 频域分析
    - 无失真传输条件为:
    - 幅度响应为实常数

实际中信道特性并非完全满足恒参,产生失真:

  • 频率失真 : 振幅~频率特性不良引起
    • 波形畸变=>majianchuanrao(传输数字信号是
  • 相位失真 : 相位~频率希特)
    • 对语音影响不打,对数字信号影响大
    • 解决方发:同上
  • 非线性失真 :
    • 元器件特性不理想
    • 定义
      输入电压~输出电压关系非线性
  • 其他失真
    • 陪你率便宜(振荡器误差),相位抖动(振荡器的频率不稳定)

变参信道的影响

- 变参信道 : 又称时变信道(或随参信道),信道参数随时间而变
- 变参信道举例 : 无线信道,如天波,地波传输,某些视距传输,各种散射无线信道
- 变参信道的特性
    - 衰减随时间变化
    - 时延随时间变化
    - 多径效应 : 信号经过几条路径达到接收端,而且每条路径的长度(时延)额衰减都随时间耳边,即存在多径传播现象

信道中的噪声

  • 噪声的定义
    • 存在的不需要的…
  • …

  • 热噪声

    • 来源 来自一切电阻性元器件中电子的热运动
    • 频率范围 : 均匀分布在大约0~10^12Hz
    • 热噪声电压有效值
      V = genghao4kTRB (V)
    • 式中
      k = 1.38 x 10^-23 (j/k) - 波尔兹曼常数
      T - 热力学温度(K)
      R - 阻值(Ω)
      B - 带宽(Hz)
    • 性质: 高斯白噪声 : 在一般通信系统工作频率范围内热噪声的频谱是均匀分布(功率谱密度在所有频率上是一常数),好像白光的频谱在可见光的频谱范围内均匀分布一样,称为白噪声.其(噪声信号时域表达式)统计特性为高斯分布

  • 分类

    • 脉冲噪声
    • 窄带噪声
    • 起伏噪声

  • 窄带高斯噪声

    • 带限白噪声:经过接收机带通滤波过滤的热噪声漫步在是白噪声,是带限(窄带)的白噪声
    • 窄带高斯噪声 :
    • 噪声功率
  • 噪声等效带宽 :(为了计算性噪比影响),等效为一个矩形

信道容量(考试考呢!)

定义 :信道的最高信息传输速率 C = R_bmax

香农公式:
带宽有限,平均功率有限的高斯白噪声连续信道(适用条件),其容量为:
C = Blog_2(1+ S/N) (bps)

S : 信号功率
B : 信道带宽
N : 信号带宽内的噪声功率;设噪声单边功率谱密度为n_0(w/Hz),则N = n_0B 因为:双边功率谱密度为n_0/2(w/Hz),故 : N = (n_0/2)XBX2=n_0B
S/N : 信噪比

通信系统主要性能指标

通信系统的基本性能指标

  • 有效性
    • 指在给定信道内能传输的信息的多少
    • 模拟通信系统:用信号的有效传输频带来度量
    • 数字通信系统:用传输速率和频带利用率来度量
    • 频带 : 带宽
  • 可靠性
    • 指接受信息的准确程度
    • 模拟通信系统:用接收端解调输出信噪比来度量
    • 数字通信系统:用误比特率或误符号率来度量

      模拟通信系统的基本指标

  • 有效性: 频带的宽度模拟小孩
  • 可靠性:

    数字通信系统的基本指标

  • 有效性:传输速率,用比特率R_b或波特率R_B表示
    • 比特率: bit/s(bps)
    • 波特率:每秒内系统传送的符号数(码元数),单位为波特(baud)
    • 关系 R_b = R_Blog_2M,一个码元用M个比特表示,
    • 利用率: 单位带宽的传输速率,即μ=R_B/B(B/Hz)或μ_b=R_b/B(bps/Hz)
  • 可靠性:误比特率P_b或误符号率P_e
    • 二进制传输中 P_b = P_e
